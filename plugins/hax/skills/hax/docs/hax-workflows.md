# HAX Workflows

## Create pages

1. Inspect the site.
2. Determine page titles and structure.
3. Generate useful starter content.
4. Use HAX CLI to add pages.
5. Verify with site inspection commands.

## Add an assessment to a page

1. Read the page content.
2. Generate assessment questions based only on that page.
3. Select the most appropriate HAX assessment component.
4. Insert the component near the end of the page.
5. Verify content and markup.

## Create a course

1. Create overview page.
2. Create module or lesson pages.
3. Add media or interactive elements where useful.
4. Add assessment/check-for-understanding pages.
5. Verify the site structure.

## Analyze a site

1. List or inspect site items.
2. Identify content coverage.
3. Identify missing pages.
4. Suggest page additions.
5. Suggest HAX components that could improve the content.
